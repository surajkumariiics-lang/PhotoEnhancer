3. **Upload image** - Drag & drop or click to browse

5. **Enhance** - Click "Enhance Image" button

## Troubleshooting

### "Application won't start"
- Check Windows Defender/antivirus settings
- Try running as administrator
- Ensure you have 2GB+ free disk space

### "Browser doesn't open"
- Manually go to: http://127.0.0.1:8000
- Check if another application is using port 8000
- Try a different browser

### "Enhancement fails"
- Ensure enough RAM (4GB+ recommended)
- Try smaller images first
- Check available disk space

### "Slow performance"
- Close other applications to free RAM
- Use smaller images for faster processing
- Consider upgrading to 8GB+ RAM
